# JSLog
JSLog is a small JavaScript library for logging you data. 

```javascript
var txt = "JavaScript syntax highlighting", 
    a = 25,
    b = false, 
    c = "other text";
log(txt, a, b, c);
```

![alt tag](http://jaroo.eu/jslog.jpg)


Forget about:
```javascript
var txt = "JavaScript syntax highlighting", 
    a = 25,
    b = false, 
    c = "other text";
console.log(txt);
console.log("a: ", a);
console.log("b: ", b);
console.log("c: ", c);
```
